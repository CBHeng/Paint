## Why can I do ?  <br />
想把 Canvas Javascript 使用更扎實，自我刻意練習的 Side Project，。<br />
刻意練習重點:
* 使用 Javascript ES6
* 使用 HTML5 Canvas
* 使用 CSS Boostrap
* 不使用JS框架，以修練 "設計模式" 為主。
* 程式碼中盡量保持 DRY (Don't repeat yourself) 的初衷
